# Core Skills

Universal capabilities applicable to any domain or project.

## What Goes Here

Skills in this directory are **fundamental, domain-agnostic capabilities** that apply across all work:

- Problem-solving frameworks
- Research methodologies
- Analytical thinking approaches
- Writing and communication patterns
- Learning strategies
- Critical thinking frameworks

## Examples

- `problem-solving.md` - Systematic approach to breaking down complex problems
- `research.md` - Research and information gathering methodology
- `writing.md` - Technical and business writing patterns
- `analysis.md` - Analytical frameworks and approaches
- `communication.md` - Communication strategies for different contexts
- `decision-making.md` - Decision-making frameworks
- `learning.md` - How you learn new concepts and skills

## When to Use Core Skills

These skills are typically combined with technical, business, or domain skills to solve specific problems.

**Example:** Research skill + API design skill = Researching and implementing a new API pattern
